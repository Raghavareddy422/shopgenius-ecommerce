Problem vs Solution

Bridging the Gap in Modern Retail

The E-Commerce Landscape Challenges

  • Static Pricing Model: Fixed retail prices fail to capture price-sensitive buyers, lowering overall checkout conversion rates.
  • High Cart Abandonment: Nearly 70% of shopping carts are abandoned without proactive recovery hooks or targeted pricing.
  • Payment Fraud & Bots: Standard checkout flows remain vulnerable to automated attacks, credentials stuffing, and credit card testing.
  • Framework Frontend Bloat: Heavy client bundles slow down initial rendering, increase network traffic, and drag down SEO rankings.

The ShopGenius Solution

  • AI Price Negotiation: Interactive sidebar bidding assistant gamifies checkouts, converting passive sessions into sales.
  • Smart Analytics Logs: Proactive cart recovery detection and automated price drop subscription alerts.
  • Algorithmic Fraud Scoring: Real-time behavioral evaluations that flag or block high-risk transactions instantly.
  • Vanilla Performance Stack: Pure HTML5, CSS3, and native JavaScript for near-instant load speeds.

Security & Risk

Algorithmic Fraud Scoring

Real-Time Behavioral Checks

  • Velocity Triggers: Analyzes session request rates to block credential testing and automated crawler bots.
  • Transaction Auditing: Evaluates order amounts against normal user averages, holding outliers for administrative authorization.
  • Telemetry Analysis: Tracks conflicting location parameters and session logs to detect account takeover attempts.
  • Stateless Authentication: Secures user APIs with JWT, refresh token rotation, and Role-Based Access Control (RBAC).

LOW RISK

Order cleared. Dispatched to automated verification.

MEDIUM RISK

Order flagged. Requires admin panel manual approval.

HIGH RISK

Transaction blocked. Account session suspended.

Backend Infrastructure

High-Performance Java & Spring Boot Core

Java 21 & Spring Boot 3.x

Leverages modern syntax and Spring virtual threads. Uses a highly modular package-by-feature file structure to ensure clean architecture boundaries.

PostgreSQL & Hibernate

Uses Spring Data JPA. Relational schemas, constraints, and indexes are managed through Flyway versioned migrations for zero-downtime database updates.

Redis Session Cache

Caches volatile active carts, user session details, and blacklisted JWT tokens to offload PostgreSQL query bottlenecks.

Unified API Exception Layer

A centralized global exception controller wraps errors into generic JSON payloads, mapping code issues to precise HTTP status codes.

Frontend Architecture

Local Development

Consistent Containerized Services

COMPILER

Maven Build

Compiles Spring Boot 3 dependency packages, runs unit tests (JUnit 5, Mockito), and packs files into lightweight executable JARs.

PRIMARY DATA

PostgreSQL Node

Isolated database storing persistent users, products, category filters, logs, address records, and checkout entries.

SESSION CACHE

Redis Cache Node

High-throughput memory store handling temporary active cart states, preventing redundant database disk operations.

DOCKER COMPOSE LOCAL ORCHESTRATION

Both database and caching services are managed locally using a single docker-compose.yml file. This ensures that the local developer environment exactly mirrors the production hosting structure.
